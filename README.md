# CMake
A CMake build script example

## Usage
### Building project
You need to have the [CMake][10] installed. Minimum Version required is 3.0.0.

In the project root directory execute the shell scrpit configure.sh with root privileges, e.g.:
```
sudo ./configure.sh 
```
